# Chatbot com Python e Gemini
Projeto simples mas que marca o inicio do meu aprendizado em IA. Um chatbot desenvolvido em Python utilizando a API do Gemini.

## Descrição

Este projeto demonstra a integração de um modelo de Inteligência Artificial generativa em uma aplicação Python.
O chatbot permite que o usuário envie perguntas e receba respostas do modelo de IA em tempo real.

## Tecnologias utilizadas

- Python
- API Gemini
- Google Colab

## Funcionalidade

- Recebe perguntas do usuário
- Envia prompts para o modelo de IA
- Retorna respostas geradas pelo modelo
- Loop de conversa até o usuário digitar "fim.
