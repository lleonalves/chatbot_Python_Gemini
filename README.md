# Chatbot com Python e Gemini
Projeto simples mas que marca o inicio do meu aprendizado em IA. Um chatbot desenvolvido em Python utilizando a API do Gemini.

## Descrição

Este projeto foi criado durante à aula do curso de Especialista de IA pela plataforma Alura. Consiste na integração de um modelo de Inteligência Artificial generativa em Python.
O chatbot permite enviar perguntas e receber respostas em tempo real.

## Tecnologias utilizadas

- Python
- API Gemini
- Google Colab

## Funcionalidade

- Recebe perguntas do usuário
- Envia prompts para o modelo de IA
- Retorna respostas geradas pelo modelo
- Loop de conversa até o usuário digitar "fim.
